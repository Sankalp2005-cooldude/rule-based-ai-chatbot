# Rule-Based AI Chatbot 🤖

A simple rule-based chatbot built using Python that responds to predefined user inputs using dictionary-based lookup and control flow logic.

## Features

* Handles greetings
* Responds to predefined commands
* Continuous chat loop
* Exit functionality
* Input sanitization using `lower()` and `strip()`
* Dictionary-based response mapping
